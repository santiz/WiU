Project Name: WorldSurf

Project Description

By the time you complete this exercise you will have learned what usability testing is, and how to use a web based usability testing solution. This solution will help you to build and share usability tests to test participants, and stakeholders with a vested interest in the creation of usable interfaces for any project. Additionally, you will build and deploy a web standards compliant landing page. This page will act as a portal, via hyperlinks, to all usability tests and the test results. This landing page will make all test and results available to stakeholders.
